# TP Node.js – Introduction

Ce projet vous guide dans les étapes essentielles pour découvrir Node.js, créer un serveur simple, manipuler des fichiers, développer une API et utiliser Express.js.

---

## Étape 1 : Installation de Node.js

1. Aller sur le site officiel : nodejs.org  
2. Télécharger la version LTS  
3. Installer Node.js selon votre système  
4. Vérifier l'installation :
```
node -v
npm -v
```
---

## Étape 2 : Création du projet

1. Créer un dossier pour votre projet  
2. Initialiser le projet avec :
```
npm init -y
```
3. Un fichier package.json est créé automatiquement

---

## Étape 3 : Création d’un serveur HTTP

Vous mettrez en place un serveur Node.js simple capable de répondre à une requête HTTP.

---

## Étape 4 : Manipulation de fichiers

Utilisation du module fs pour lire un fichier texte et afficher son contenu via votre serveur.

---

## Étape 5 : Création d’une API simple

Mise en place :

- d'une route pour obtenir la liste des utilisateurs  
- d'une route pour consulter un utilisateur via son identifiant  
- d'une gestion simple des erreurs

---

## Étape 6 : Utilisation d’Express.js

Installation et introduction à Express pour :

- organiser les routes
- envoyer des réponses JSON
- simplifier le code du serveur
- gérer les requêtes plus efficacement

---

## Étape 7 : Ajout de fonctionnalités avancées

Intégration de modules supplémentaires :

- dotenv pour gérer les variables d'environnement
- morgan pour la journalisation des requêtes

Mise en place :

- d'un dossier public pour servir une page HTML
- d'une API Express structurée
- d'une gestion d'erreurs plus propre

---

## Conclusion

Ce TP permet d'apprendre :

- l'installation et la configuration de Node.js
- la création d'un serveur HTTP
- la manipulation de fichiers
- la construction d’une API REST
- l'utilisation d'Express.js
- la gestion des middlewares, des routes et des variables d'environnement
- la mise en place d'une interface web simple
